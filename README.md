# Household-Electric-Power-Consumption-data-in-JAVA
An programm in JAVA which take data from a file with data from 
https://www.kaggle.com/uciml/electric-power-consumption-data-set 
and compute and print to files chosen by the user the avg and sum consumption for the place of house like kitchen etc.
In the resources there are files to run and see the results.
To run it :run the gui.java file.
